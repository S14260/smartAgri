// 项目简介
从卫星数据到巡田决策的完整闭环

三层分离架构，前端可视化、Java 业务逻辑、Python 遥感计算各司其职。管理员通过自然语言与 AI Agent 交互，自动完成异常检测与路径规划。

🏗️

三层分离架构

前端 LayUI + CesiumJS 负责可视化，Spring Boot 处理业务逻辑与认证，Flask 微服务专注遥感计算与空间分析。

📡

多源遥感处理

支持 NDVI / EVI / SAVI / NDWI 四种光谱指数，自动坐标重投影、矢量裁剪、Lanczos 超分辨率与边缘抗锯齿。

🧠

ReAct AI Agent

基于 DeepSeek LLM 的推理-行动循环，5 个自动注册工具，SSE 流式输出思考过程与工具调用状态。

核心功能模块

地块管理、遥感处理、异常检测、路径规划、AI Agent、数据驾驶舱、GPS 打卡 — 覆盖农业遥感监测全链路。

🗺️

地块管理 + AI 自动圈地

支持 Leaflet.draw 手动圈地与 YOLO11-Seg 自动圈地，结合坐标转换、拓扑简化、面积计算和高德逆地理编码，实现地块快速建档。

🤖

ReAct AI Agent

自然语言驱动的智能巡田。DeepSeek LLM 推理 + 5 个自动注册工具，SSE 流式输出 6 种事件类型，实时展示思考链。

巡田任务工单生成

基于异常检测结果自动生成巡田工单，关联异常区域、规划路线、分配巡检人员，支持工单状态跟踪与 LLM 智能报告。
