// 04 智能路径规划
异常驱动的巡田路线优化

基于异常权重的地块优先级排序，最近邻初始路径 + 2-opt 局部优化，Haversine 球面距离计算。

异常权重计算

综合异常优先级、严重程度与面积，加权计算每个地块的巡田权重。

priority × 0.5 severity × 0.3 area × 0.2

最近邻初始路径

从最西端点出发，贪心选择最近未访问地块，快速生成初始可行路径。

Nearest Neighbor West-most Start

2-opt 局部优化

迭代 200 次交换路径段，消除交叉、缩短总距离。Haversine 公式计算球面距离。

2-opt × 200 Haversine

结果输出

GeoJSON 格式路径数据、可视化 PNG 图像、量化评估报告，前端 Leaflet / CesiumJS 可视化。

GeoJSON PNG Report
// 05 AI Agent 交互演示
自然语言驱动的智能巡田

用户用自然语言提问，Agent 自动推理意图、调用工具、返回结构化结果。SSE 流式输出思考过程。
